Git

  • init - расшифровывается как "Create an empty Git repository or reinitialize an existing one" — создание репозитория.
  • clone - расшифровывается как "Clone repository" - клонирование удаленного репозитория.
  • commit - расшифровывается как "Record changes to the repository"- запись изменений в репозиторий.
  • push - расшифровывается как "Update remote refs along with associated objects" - отправка изменений локального репозитория в удаленный репозиторий.
  • status - расфшифровывается как "Show the working tree status" - просмотр списка измененных файлов.
